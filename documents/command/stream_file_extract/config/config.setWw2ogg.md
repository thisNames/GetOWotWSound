# 设置项目配置 ww2ogg（子命令）

## 命令
- `-cfg-w2g`, `-config-ww2ogg` 

## 描述
- 设置 SoundMod/ww2ogg-v0.24.exe 工具路径, `ww2ogg = <path>`

## 文档
```txt
将 .wem 转换成 .ogg 工具路径
示例：owo -cfg-w2g "E:\Developer\Projects\GetOWotWSound\SoundMod\ww2ogg-v0.24.exe"
```
## 父命令
- `-cfg`, `-config`, [config.md](config.md)
